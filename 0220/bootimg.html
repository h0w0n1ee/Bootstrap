04_부트스트랩(이미지관련)



1. 모서리 둥글게 :: class="rounded"



2. 모서리 둥글게(원모양) :: class="rounded-circle"



3. 이미지를 썸네일 이미지처럼 보이게 :: class="img-thumbnail"

.img-thumbnail :: border, padding, border-radius 속성값 사용


4. 이미지와 문자를 사용하여 콘텐츠 제작시 :: figcaption, figure

이미지를 반응형으로 동작하게 만드는 클래스 :: class="img-fluid" 적용시, width:100% / height:auto 속성이 자동으로 들어감!!

부모의 크기가 변하려면 이미지 크기도 같이 반응!
