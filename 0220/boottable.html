class="table" :: 테이블 기본서식

class="table-striped" :: 행마다 다른 색상 적용

class="table-bordered" :: 테이블 셀 테두리 만들기

class="table-hover" :: 마우스 오버시 배경색 변경

class="table-dark" :: 테이블 제목줄(thead)에 배경색 어둡게 적용

class="table-success", class="table-danger", class="table-info", class="table-primary" :: tr, td에 각각 분류 색상을 적용 할 수 있음

class="table-responsive" :: 화면의 너비가 768px미만일 경우, 가로로 스크롤 됨!
