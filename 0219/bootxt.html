02_부트스트랩(글자, 문단관련)

부트스트랩에서는 기본 글자 크기가 16px이고, 행 높이(줄간격)는 1.428이 기본값이다.

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본값이다.

mark :: 글자에 하이라이트 효과를 적용함. 부트스트랩 적용 안할시, 노란배경으로 나타남!
abbr :: 약어나 줄임말을 사용하고자 할 때 사용함. (16px)

★ 텍스트의 중요도 ★

☞ 글자 색상을 다르게 표시하기

text-muted :: 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (가장 나중순위 색깔은 회색!)

text-primary :: 텍스트를 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (가장 우선순위 색깔은 파란색!)

text-success :: 텍스트를 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (초록색)

text-info :: 텍스트를 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (하늘색)

text-warning :: 텍스트를 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (주의표시는 주황색!)

text-danger :: 텍스트를 중요도에 따라 문자의 색상을 다르게 표시할 수 있다. (위험, 매우중요한 표시는 빨간색!)

★ 문단 관련 서식 ★

class=lead :: 단락을 눈에 띄게 하기 (글자크기_20px)

class=small :: 상위 문자 크기의 85%(=14px)로 문자 크기를 작게 설정함

class=text-start :: 문단 왼쪽 정렬

class=text-end :: 문단 오른쪽 정렬

class=text-center :: 문단 중앙 정렬

class=text-justify :: 문단 양쪽 배분 정렬

class=text-nowrap :: 문단 끝에서 글자가 다음줄로 넘어가지 않게 하기

class=uppercase :: 문자를 대문자로 출력

class=lowercase :: 문자를 소문자로 출력

text-capitalize :: 영문자의 첫 글자만 대문자로 출력
